# sd-webui-installation-windows

การติดตั้งโปรแกรม Stable Diffusion By SalmonRK
การใช้งาน เบื้องต้น

Folder 2 ส่วน
1. stable-diffusion-webui จัดเก็บโปรแกรมการใช้งาน ถ้าโปรใช้งานไม่ได้ แนะนำให้ลบ folder แล้วรัน สคริปติดตั้งใหม่
